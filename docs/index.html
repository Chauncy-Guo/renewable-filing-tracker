全国新增建档立卡新能源装机

数据采集与汇总工具 — 项目文档

项目简介

自动化采集 国能新能建档立卡系统 发布的《全国新增建档立卡新能源发电项目情况》公告, 把随附的 PDF / Word 文件解析为 Excel, 并按月份和类型汇总到统一表格。

项目文档

🏗️架构设计

模块划分、调用链、扩展点; 数据流图。

📊数据格式

输入公告、PDF、Word 与输出 Excel 的字段定义。

❓常见问题

文件头校验、相对路径、动态表类型、错误修复等。

快速开始

# 1. 安装依赖
npm install
pip install -r requirements.txt

# 2. 一键执行 (下载 → 提取 → 汇总)
node scripts/download_new.js
python scripts/extract_all.py
python scripts/merge_excel.py
